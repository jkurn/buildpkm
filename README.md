# BuildPKM

**Stop hoarding. Start compounding.**

A free Digital Hoarding Assessment that scores your knowledge management system across three pillars — Capture, Processing, and Retrieval — and gives you a personalized action plan.

## Live Site

[buildpkm.com](https://buildpkm.com)

## How It Works

1. **Landing page** — identifies the digital hoarding problem and drives to the quiz
2. **15-question assessment** — 10 scored questions across 3 PKM pillars + 5 qualifying questions
3. **Dynamic results** — animated score, pillar breakdown, personalized insights, and segmented CTA based on user profile

## Stack

- Static HTML/CSS/JS (no dependencies)
- GitHub Pages
- Custom domain via CNAME
